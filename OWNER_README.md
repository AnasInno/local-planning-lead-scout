# Planning Lead Scout — double-click version

1. Unzip the folder.
2. On macOS, double-click `Start.command`. On Windows, double-click `Start.bat`.
3. Paste your Gemini API key once when asked.
4. Enter your trade, town or postcode, and date range.
5. Click **Find leads**.
6. Download the CSV or print the shortlist.

Notes:
- `Manchester` runs fake demo planning records included in the folder.
- Postcode or outcode examples such as `M14` use public planning data.
- No scraping, no CRM, no email sending.
